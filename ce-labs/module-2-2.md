## Objective(-s):
- Create a VDC.
- Remove the "users" group from the "default" VDC.

# Create a VDC.

    
## 2.2.1

Navigate to **System -> VDCs** to access the VDCs management panel. 

<img src="./../assets/ca-images/module2_lab2/s1.png" class="img_80_percent">


## 2.2.2

Press **Create** to add a new VDC.

<img src="./../assets/ca-images/module2_lab2/s2.png" class="img_80_percent">
    
## 2.2.3

Name it the way you wish. In this guide we are going to use the **prodVDC** as a reference. 

<img src="./../assets/ca-images/module2_lab2/s3.png" class="img_80_percent">

    
## 2.2.4

Select **users** from the group selector.

<img src="./../assets/ca-images/module2_lab2/s4.png" class="img_80_percent">

    
## 2.2.5

Select the **default** cluster to cover all resources that we currently have. 

<img src="./../assets/ca-images/module2_lab2/s5.png" class="img_80_percent">

    
## 2.2.6

Leave Custom Variables empty and press **Finish**.

<img src="./../assets/ca-images/module2_lab2/s6.png" class="img_80_percent">
    

# Remove the "users" group from the default VDC.


## 2.2.7

Select the **default** VDC and press **Update**.

<img src="./../assets/ca-images/module2_lab2/s7.png" class="img_80_percent">

    
## 2.2.8

Go to the **Select a group** tab and make sure to remove the **users** from the list.

<img src="./../assets/ca-images/module2_lab2/s8.png" class="img_80_percent">

    
# Leave rest as is and save the changes.
